
# E-commerce

Simple and functional e-commerce site with product display, product list and shopping cart functionality. In addtion all products are dynamically populated with a JSON file.



## Installation

First clone and go to e-commerce and install the following thing.

```bash
  npm install 
```
    
## Deplyment

To run the e-commerce follow this code

```bash
  npm run dev
```

To run the json file follow this code

```bash
  npx json-server --watch --port 5000 products-list.json
```
